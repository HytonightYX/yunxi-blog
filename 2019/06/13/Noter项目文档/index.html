Noter-文档

Posted on 2019-06-13 Views: Valine:

想法&相关介绍

  • 作为一名IT学习者，向来以开源为荣。每逢期末就有种分享自己的笔记的冲动，对于弱势学科也特别想要看看学霸们的笔记。因此萌生了这个知识共享平台的想法
  • 自己平时喜欢用markdown来写点文章，真心觉得这是一种很高效的记录方式，而且结构统一，书面美观，对工科生来说非常好用
  • 在网上找了一番，感觉同类产品比较少。类似的比如腾讯在线文档这类产品，感觉不够轻量，而且文档类型大多是巨硬的office，后来找到了Tower这一款产品，感觉比较符合我的预期，看起来很清爽，所见即所得，不过同样的，功能太多，产品定位也略有不同，但可以以其为蓝本，精简功能

API

全部采用Restful风格,数据传输采用JSON格式

笔记模块

功能 请求类型 路径 参数类型
笔记测试接口 GET api/notes/test
所有笔记 GET api/notes
添加笔记 POST api/notes JSON
根据id获取笔记 GET api/notes/:id
更新笔记 PATCH api/notes JSON
根据所有者id获取他的所有笔记 GET Api/notes/owner/:ownerId
删除笔记 DELETE api/notes/:id

用户模块

功能 请求类型 路径 传输类型
用户测试接口 GET api/users/test
获取用户详情 GET api/users/:id
新用户注册 POST api/users/register 暂时废弃
修改用户信息 PATCH api/users/:id {newUser}

系统功能

用户个人模块

  • 修改个人信息
  • 使用Github账户一键登录

笔记仓库模块

  • 对属于自己的笔记的增、删、改操作
  • 查看笔记

笔记市场模块

  • 查看所有用户共享的笔记
  • 点赞

布局
