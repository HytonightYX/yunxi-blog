[Leetcode] 152改——乘积最大连续子串

 2019-05-10  2019-05-23   5 minutes read (About 675 words) 0 visits

给定一个浮点数数组 nums ，找出一个序列中乘积最大的连续子序列（该序列至少包含一个数）。可以使用动态规划解题。

示例 1:

输入: [2,3,-2,4]
输出: 6
解释: 子数组 [2,3] 有最大乘积 6。

示例 2:

输入: [-2,0,-1]
输出: 0
解释: 结果不能为 2, 因为 [-2,-1] 不是子数组。

一、暴力求解

/**
 * 暴力求解，两个循环轮询
 * 时间复杂度：O(n^2)
 * @param arr 输入数组
 * @return ret 最大值
 */
public double prodSubArray(double[] arr) {
   double ret = arr[0];

   for (int i = 0; i < arr.length; i++) {
      double x = 1;
      for (int j = i; j < arr.length; j++) {
         x *= arr[j];
         ret =  Math.max(ret, x);
      }
   }
   return ret;
}

二、动态规划

重点就是这个负数：

你想想，可能上一次是最大值，这次乘了个负数结果就变最小了，上次是个最小值，乘了个负数结果又变最大了。所以结果可能会在最大值与最小值之间跳变的，所以我们要将最大值与最小值都存下来！！

/**
 * 动态规划(DP)
 * 时间复杂度：O(n)
 * @param arr 输入数组
 * @return ret 最大乘积
 */
public double dpProdSubArray(double[] arr) {
    // 记录最大乘积
    double maxProd = arr[0];
    // 记录最小乘积
    double minProd = arr[0];
    double ret = arr[0];

    for (int i = 1; i < arr.length; i++) {
        double prod1 = maxProd * arr[i];
        double prod2 = minProd * arr[i];
        maxProd = Math.max(Math.max(prod1, prod2), arr[i]);
        minProd = Math.min(Math.min(prod1, prod2), arr[i]);
        ret = Math.max(ret, maxProd);
    }

    return ret;
}

三、完整代码

public class Main {
   /**
    * 暴力求解，两个循环轮询
    * 时间复杂度：O(n^2)
    * @param arr 输入数组
    * @return ret 最大值
    */
   private static double prodSubArray(double[] arr) {
      double ret = arr[0];

      for (int i = 0; i < arr.length; i++) {
         double x = 1;
         for (int j = i; j < arr.length; j++) {
            x *= arr[j];
            ret =  Math.max(ret, x);
         }
      }
      return ret;
   }

   /**
    * 动态规划(DP)
    * 时间复杂度：O(n)
    * @param arr 输入数组
    * @return ret 最大乘积
    *
    * 注意点就是因为存在负数:
    *   可能一个临时最大值，乘了个负数结果就变最小了，一个临时最小值，乘了个负数结果又变最大了
    *   所以结果可能会在最大值与最小值之间跳变的，所以我们要将最大值与最小值都存下来！！
    */
   private static double dpProdSubArray(double[] arr) {
      // 记录最大乘积
      double maxProd = arr[0];
      // 记录最小乘积
      double minProd = arr[0];
      double ret = arr[0];

      for (int i = 1; i < arr.length; i++) {
         double prod1 = maxProd * arr[i];
         double prod2 = minProd * arr[i];
         maxProd = Math.max(Math.max(prod1, prod2), arr[i]);
         minProd = Math.min(Math.min(prod1, prod2), arr[i]);
         ret = Math.max(ret, maxProd);
      }

      return ret;
   }

   public static void main(String[] args) {
      double[] arr = {-2.5, 4, 0, 3, 0.5, 8, -1};
      System.out.println("输入数组: " + Arrays.toString(arr));
      System.out.println("暴力求解: " + prodSubArray(arr));
      System.out.println("动态规划: " + dpProdSubArray(arr));
   }
}
